# 🧠 Neural Vibe Engine (V5.0)

**Highly accurate. Mildly calling you out.**

The Neural Vibe Engine is a sophisticated, zero-backend personality mapping tool. Unlike static quizzes, it uses an adaptive entropy-reduction algorithm to select questions in real-time based on your previous answers, aggressively narrowing down your psychological archetype.

## 🚀 Core Features

- **Adaptive Probing**: The quiz path is dynamic. Every choice changes the pool of potential next questions to challenge your dominant traits.
- **Zero-AI / Zero-Cloud**: All processing is handled locally in your browser. No data leaves your device.
- **Neural History**: Built-in `localStorage` integration tracks your "Vibe Evolution" across multiple sessions.
- **Achievement System**: Unlock unique "Digital Badges" based on rare or extreme trait combinations.
- **Instant Sharing**: Export your "Neural ID" via URL-encoded vectors. Share your vibe without needing a database.
- **Data Visualization**: Real-time Radar Mapping of your personality dimensions (Chaos, Logic, Emotion, Imagination, Order).

## 🛠️ Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS + ShadCN UI (Brutalism Lite aesthetic)
- **Visualization**: Recharts (Radar Data Mapping)
- **Icons**: Lucide React
- **Type Safety**: TypeScript + Zod

## 🛡️ Privacy

Your data is yours. This app uses no external APIs, no tracking cookies, and no AI models. It is a pure logical engine designed for high-fidelity personality assessment.
